How it works:

When bootstrapping the app configService will get the `env.json`file from `assets/env/env.json` and try to get the conf from an external URL described in serverUrl property, ex: http://localhost:9090/front/conf.
If the serverUrl property is empty then it will look at an embedded file which is declared inside the property `embeddedConfFileUrl`.
